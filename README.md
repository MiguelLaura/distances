# CLI distance

CLI built during an initiation week to DevOps, Software engineering, Data engineering.

## How to use

```bash
pip install .
```

Either as a package:
```bash
from distances import hausdorff, minimal 
```

Or as a CLI:
```bash
distance_computor
```

To test the code:
```bash
pip install pytest
pytest
```

To use black:
```bash
pip install black
black distances
```
